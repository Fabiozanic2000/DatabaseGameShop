READ ME FILE GRUPPO 1
-Verisione DBMS utilizzata:PostgresSQL 13.2;
-Per installare il database creato dal nostro gruppo
è necessario il file generato da pg_dump che allegheremo 
al nostro progetto. Creare un nuovo database all'interno di postgres
premere con tasto destro sul database e selezionare l'opzione di 
restotre e il file sopra menzionato.
-Nome file pg_dump: Progetto_backup.

-In altrnativa al file pg_dump è anche disponibile il codice
sorgente sql utilizzato dal gruppo per la creazione delle tabelle 
e il popolamento delle stesse.
